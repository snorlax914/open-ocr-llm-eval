a. 기존 카테고리와 전혀 상관 없는 문서를 입력했을 때 맥락에 맞게 새 이름을 잘 지어내는지 확인하기.

b. 2주차에 사용한 샘플 외에 의도적으로 분류가 모호한 텍스트를 포함한 데이터셋을 구축하여 성공률 측정하기.
