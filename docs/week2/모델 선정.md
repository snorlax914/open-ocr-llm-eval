# 한국어/비즈니스 특화 LLM 모델 추천

## 1. 최우선 추천 모델 (한국어/비즈니스 특화)

### ① EXAONE 3.5 7.8B Instruct (LG AI Research)

LG에서 개발한 최신 모델로, 한국어와 영어 성능의 균형이 매우 뛰어납니다.

- **특징**: 한국어 데이터 학습 비중이 압도적으로 높으며, 비즈니스 용어 및 행정 서식에 대한 이해도가 높습니다.
- **분류 적합성**: OCR 텍스트 내의 복잡한 표 구조나 행정 문구에서 핵심 키워드를 추출하여 '사업자등록증', '인감증명서' 등을 정확히 구분해 내는 능력이 탁월합니다.
- **장점**: Qwen2.5-7B의 강력한 대안으로, 한국어 뉘앙스 파악 면에서 더 우세할 수 있습니다.

### ② HyperCLOVA X SEED 8B (Naver)

네이버의 HyperCLOVA X 엔진을 기반으로 한 경량화 모델입니다.

- **특징**: 국내 행정, 법률, 문화적 맥락을 가장 잘 이해하는 모델입니다. '인감증명서'나 '등기부등본' 같은 한국 고유의 문서 체계를 분류하는 데 있어 가장 강력한 Zero-shot 성능을 보입니다.
- **분류 적합성**: OCR 결과물에 오타가 있더라도 문맥을 통해 해당 문서가 무엇인지 유추하는 '맥락 파악 능력'이 뛰어납니다.

## 2. 범용 성능 및 효율성 모델

### ③ Llama 3.1 8B Instruct (Meta)

전 세계적으로 가장 많이 사용되는 표준 모델입니다.

- **특징**: 한국어 전용 모델은 아니지만, Llama 3.1로 넘어오면서 다국어 성능이 비약적으로 상승했습니다.
- **장점**: 에코시스템이 매우 잘 갖춰져 있어 vLLM, Ollama 등 다양한 추론 엔진에서 최적화된 속도를 보장받을 수 있습니다.
- **분류 적합성**: 지시 이행(Instruction Following) 능력이 매우 정교하여, "JSON 형식으로만 답하라"는 식의 제약을 가장 잘 지킵니다.

### ④ Mistral NeMo 12B Instruct (Mistral / NVIDIA)

Mistral과 NVIDIA가 협업하여 만든 12B 파라미터 모델입니다.

- **특징**: 7~8B 모델들보다 파라미터가 조금 더 많아 추론의 깊이가 다릅니다. (하지만 Thinking 모델은 아닙니다.)
- **장점**: 컨텍스트 윈도우가 크고 텍스트 압축률이 좋아, OCR 결과물이 매우 길거나 여러 페이지인 문서를 처리할 때 정보 손실이 적습니다.
- **분류 적합성**: 단순 분류를 넘어 문서 내 주요 필드(성명, 번호 등)를 함께 추출하는 태스크를 동시에 수행할 때 안정적입니다.
