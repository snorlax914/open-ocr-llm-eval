# 1주차 과제

## 주제
문서 OCR을 위한 LLM, OCR 모델 탐색 및 성능 평가

---

## 1. (필수) 오픈소스 기반 LLM 및 OCR 모델 탐색

- 개인 컴퓨터 또는 Google Colab 환경에서 직접 구축하여 사용할 수 있는 모델을 조사

### 참고 사항
- LLM 예시:
  - Gemma
  - Qwen
- OCR 모델 예시:
  - PaddleOCR
  - Tesseract

### 주의 사항
- 각 모델의 버전 차이를 고려하여 조사
  - 예:
    - PaddleOCR-VL-1.5
    - PP-OCRv5

---

## 2. (필수) OCR 성능 평가 기준 설정

- 조사한 모델을 바탕으로 평가 기준 정의

### 예시 기준
- 한국어 정확도
- 영어 정확도
- 레이아웃 구분 능력
- 페이지 당 분석 속도

---

## 3. (선택) OCR 성능 평가 수행

- 선정한 모델과 평가 기준을 기반으로 실제 성능 평가 진행

### 요구 사항
- 분석 환경 명시
